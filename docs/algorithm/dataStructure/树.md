# 二叉树 Tree

- 二叉查找树（Binary Search Tree）
- 平衡二叉查找树（Balanced Binary Search Tree）
- 红黑树(Red-Black Tree )
- B-tree
- B+-tree
- B*-tree (B~Tree)


- 基本信息:
  - 名称: 二叉树
  - 作者:

## 特性

1. 每个父节点下只有两个子节点
2. 左边的子节点比父节点小，右边的子节点比父节点大

## 场景

- DOM树
- AST树
- Virtual DOM
- 查找一组数据中的最大值和最小值
- 目录文件遍历——后序遍历

- 红黑树
  - TCP 连接实现多路复用时借助的epoll(红黑树、队列)
  - Linux 进程调度、进程空间管理
  - Nginx 定义缓存 key
- B+树
  - MySQL数据库索引使用 B+ 树提高查询效率
- Tire 树
  - Linux 文件系统中为了提高目录项对象的处理效率：LRU、哈希表Linux 路由表 Tire 树

## 方法

- 添加节点 insert(value)
- 查找节点 search(value)
- 移除节点 remove(value)
- 遍历节点 traverse(value)

- 获取最小值 min()
- 获取最大值 max()
